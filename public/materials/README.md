# Materials Library

Add your published files in these folders:

- `/public/materials/slides/`
- `/public/materials/summaries/`
- `/public/materials/roadmaps/`
- `/public/materials/templates/`

Supported examples: `.pdf`, `.pptx`, `.docx`, `.xlsx`, `.csv`, `.zip`, `.md`, `.txt`.

How it works:
- Any file pushed into these folders is automatically listed on the Materials pages.
- No code edits are needed for normal content updates.
